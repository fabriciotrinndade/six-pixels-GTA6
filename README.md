# Six Pixels

**Six Pixels** is a top-down 2D pixel game inspired by **GTA 1 & GTA 2**, reimagined with a modern **neon / GTA-6-style aesthetic**.

Built entirely using **HTML5 Canvas and vanilla JavaScript**, the project focuses on clean game logic, pixel-art visuals, and an expandable city system.

This repository serves both as a **playable game prototype** and a **learning project** for understanding classic game mechanics on the web.
